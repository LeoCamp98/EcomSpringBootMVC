# EcomSpringBootMVC
Progetto base sviluppato durante lo studio del framework Spring, in cui ho utilizzato Spring Boot, Maven, PostgreSQL e JPA per realizzare una REST API.
L’applicazione è stata testata tramite il client REST Postman, verificando il corretto funzionamento degli endpoint per la gestione delle principali richieste HTTP e segue un'architettura MVC a livelli (Controller -> Service -> Repository).
